# Float Clock — นาฬิกาลอยจอ + ตัวเตือนนับเวลา

เว็บล้วน (vanilla HTML/CSS/JS) แสดงเวลาเรียลไทม์ ลอยทับจอตอนเล่นเกม/เปิดโปรแกรมอื่นด้วย
**Document Picture-in-Picture** และมีตัวจับเวลาเตือนตามที่ชอบ ใช้ธีม **light minimal** ร่วมจาก
`../_design-system/tokens.css`

## ฟีเจอร์
- **นาฬิกาเรียลไทม์** — 12/24 ชม., เปิด-ปิดวินาที, วันที่ภาษาไทย
- **ลอยจอ (PiP)** — กดปุ่ม "ลอยจอ" → หน้าต่างนาฬิกาเล็กลอยอยู่บนสุดทุกหน้าต่าง
- **ตัวเตือนนับเวลา** — ตั้งได้ทั้ง **นาที + วินาที** (preset 30 วิ / 1 / 5 / 15 / 25 / 60 น. หรือพิมพ์เอง), เริ่ม/หยุดชั่วคราว/รีเซ็ต (ขั้นต่ำ 1 วินาที)
- **รูปแบบเตือนเมื่อครบ** (เลือกได้หลายอย่างพร้อมกัน):
  - 🔊 เสียงบี๊บ (Web Audio — ไม่ต้องมีไฟล์เสียง)
  - 💡 กระพริบ/เปลี่ยนสีจอ (กระพริบทั้งหน้าหลักและหน้าต่าง PiP)
  - 🔔 Browser Notification
  - 🔁 นับซ้ำอัตโนมัติ (วนทุก N นาที แบบ Pomodoro/Pulse)
- จำค่าตั้งล่าสุดด้วย `localStorage`

## วิธีรัน
PiP **ต้องเปิดผ่าน https หรือ http://localhost** (secure context) — เปิดไฟล์ตรง ๆ (`file://`) จะใช้ PiP ไม่ได้

```bash
# จากโฟลเดอร์ ClaudeSync (root)
python -m http.server 8793
# เปิด http://localhost:8793/clock-overlay/
```
(มี config `clock-overlay` ใน `.claude/launch.json` แล้ว)

## ข้อจำกัด / Edge cases
- รองรับ Chrome/Edge ใหม่เท่านั้น (Firefox/Safari ยังไม่รองรับ Document PiP → ปุ่มจะแจ้งเตือน)
- เกมแบบ **Fullscreen exclusive** อาจบังหน้าต่าง PiP — ให้ตั้งเกมเป็น **Borderless/Windowed**
- เสียงบี๊บต้องมี user gesture ครั้งแรก (กดปุ่มในหน้าเว็บก่อน) เบราว์เซอร์ถึงเล่นเสียงได้
- Notification ต้องกดอนุญาตสิทธิ์ครั้งแรก

## ไฟล์
- `index.html` — โครงหน้า + ก้อน `#pip-content` ที่ย้ายไป-มาระหว่างหน้าหลักกับ PiP
- `style.css` — สไตล์เฉพาะหน้า (สี/ฟอนต์ดึงจาก token กลาง)
- `app.js` — นาฬิกา, ตัวจับเวลา, ตัวเตือน, ลอจิก PiP
